# Registro de Errores Semánticos y Gramaticales

| Error del Usuario | Corrección Correcta | Explicación / Matiz | Fecha |
| --- | --- | --- | --- |
| Escribir "Logré mi objetivo este mes usando 牢靠" | "我这个月实现了我的目标" (Logré mi objetivo...) o "...达到了..." | 牢靠 (láokào) es un adjetivo que significa firme, sólido o confiable. No se puede usar como verbo de acción para decir "lograr" o "alcanzar". Para ello se debe usar el verbo 实现 (shíxiàn) o 达到 (dádào). | 2026-05-27 |
| Usar 学习 como sustantivo incorrectamente | Correcto uso | Explicación breve | 2026-05-27 |
